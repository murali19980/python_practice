# 🐍 Python Practice Repository

> My personal coding gym — a structured sandbox for grinding Python syntax, standard library mastery, algorithms, and random experiments.

This repository is **not** a portfolio project. It's a living notebook where I:
- Drill core syntax and language features
- Solve DSA problems (LeetCode, HackerRank, etc.)
- Test out standard library modules
- Experiment with third-party libraries (NumPy, Pandas, Requests)
- Keep reusable snippets and one-liners for quick reference

---

## 📂 Repository Structure

| Folder | What's inside |
|--------|---------------|
| [`01_syntax_basics/`](./01_syntax_basics) | Variables, data types, strings, lists, dicts, loops, conditionals, functions |
| [`02_intermediate/`](./02_intermediate) | Comprehensions, lambdas, generators, decorators, context managers, exception handling |
| [`03_oop/`](./03_oop) | Classes, inheritance, magic methods, dataclasses, properties |
| [`04_stdlib_mastery/`](./04_stdlib_mastery) | Python standard library deep-dives (`os`, `pathlib`, `re`, `datetime`, `collections`, `itertools`, `threading`, `asyncio`) |
| [`05_algorithms_dsa/`](./05_algorithms_dsa) | Sorting, searching, linked lists, trees, graphs, dynamic programming, sliding window |
| [`06_data_science_numpy/`](./06_data_science_numpy) | NumPy, Pandas, Matplotlib/Seaborn, scikit-learn basics |
| [`07_snippets/`](./07_snippets) | Scratchpad for daily experiments, useful one-liners, API requests, logging configs |

## 🚀 How to Use This Repo

### Clone it
```bash
git clone https://github.com/murali19980/python_practice.git
cd python_practice
```

### Set up a virtual environment (optional but recommended)
```bash
python3 -m venv venv
source venv/bin/activate      # On Windows: venv\Scripts\activate
pip install -r requirements.txt
```

### Run any file
```bash
python 01_syntax_basics/lists_tuples_sets.py
```

## 📝 My Practice Routine

- **Daily warm-up**: Pick a folder, write or refactor one script.
- **LeetCode dumps**: Save problem statements as comments + solutions in `05_algorithms_dsa/`.
